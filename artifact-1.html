Artifact 1

AI Speaker Bot

An interactive AI speaker assistant combining conversational design, prompt engineering, and web development.

01

Overview

The Boyz AI Speaker Bot is an interactive course project that combines conversational AI, web development, product thinking, and visual communication. The chatbot helps users discover information about speakers through a simple and engaging interface.

02

Objective

The objective was to design an AI-supported experience that responds to user questions, presents useful speaker information, and demonstrates how prompt engineering can support an interactive digital product.

03

Process

I identified the target audience and expected questions, organized speaker-related content, designed the conversation flow, built the web interface, tested responses, and refined the experience based on usability and clarity.

04

Tools & Technologies

Artificial IntelligencePrompt EngineeringHTMLCSSJavaScriptGitHub Pages
05

Skills Demonstrated

Prompt design, conversational flow development, audience analysis, front-end development, testing, problem solving, visual communication, and project presentation.

06

Value Proposition

The project demonstrates how an AI chatbot can make information easier to access by replacing static navigation with a conversational interface. It also shows how technical and communication skills can be combined to create a complete user-facing product.

07

Reflection

I selected this artifact because it represents the practical application of concepts learned in the course. Creating the chatbot helped me understand that an effective AI solution requires more than generating responses. It also requires clear content, thoughtful prompts, a logical conversation flow, testing, and an interface that users can understand. The project improved my confidence in combining AI concepts with web technologies and showed me how user feedback can guide revisions.

08

Evidence

The live chatbot and source-code repository provide evidence of the completed application and its implementation.
